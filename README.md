#Bookly

Bookly is a virtual library app clone.
Bookly lets you to create virtual library. You can add a book/quote. All data set stored on cloud, so your books/quotes never be disappear unless you delete them.


Techs that i use:
CoreData,
FirebaseStorage, 
FirebaseAuth
